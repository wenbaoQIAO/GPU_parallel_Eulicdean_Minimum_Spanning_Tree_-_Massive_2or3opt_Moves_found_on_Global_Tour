Contenu du dossier application_template
---------------------------------------
Le dossier contient un schema type (infrastructure) d'application/fonction 
d'optimisation base sur :
- recherche locale iterative a voisinage variable
- algorithme memetique (application de la recherche locale au sein une approche à base de population de solutions)
et une librairie d'objets geometriques de base dans l'espace de nommage :
- geometry_prop

Dossiers
--------
- application : sources du programme/fonction
- doc_specifications : spécifications de départ du programme
- doc_doxygen : documentation Doxygen html generee à partir des fichiers sources
- doc_conception_generale : document de conception generale du programme

Dependances a l'execution
-------------------------
dll_Qt_4.8.4 : librairies QT sous compilation microsoft-windows
dll_Qt_4.8.4_mingw : librairies QT sous compilation minGW
dll_Qt_4.8.4_rasp : librairies QT pour cross-compilation et compilation sur ARM V6 raspberry-pi
dll_WindowsMicrosoft : librairies windows sous compilation microsoft-windows

Dependances a la compilation
----------------------------
- QT version 4 ou 5
- Boost version 1.55
